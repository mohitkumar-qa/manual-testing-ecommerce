# Manual Testing Project - OpenCart Demo E-Commerce Website

## Application Under Test

**Application:** OpenCart Demo E-Commerce Website

**URL:** https://demo.opencart.com/

**Project Status:** Environment Blocked (HTTP ERROR 522)

---

## Project Overview

This project demonstrates end-to-end manual testing of the OpenCart Demo E-Commerce web application.

The testing process was performed using an Agile-based workflow, starting from Requirement Analysis, Epic Creation, User Story Definition, Test Scenario Design, Test Case Preparation, and Defect Reporting.

Test management and documentation were maintained using Jira and Excel.

## Environment Issue Encountered

During the test execution phase, the public demo environment became unavailable and returned **HTTP ERROR 522**.

As per real-world QA practices:

* A blocker defect was raised.
* Test execution status was marked as **Blocked**.
* Documentation and deliverables were completed successfully.
