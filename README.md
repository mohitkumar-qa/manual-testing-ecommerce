# Manual Testing Project - OpenCart Demo Ecommerce Website

## Application Under Test
OpenCart Demo E-Commerce Website
URL: https://demo.opencart.com/

## Project Overview
This project demonstrates end-to-end manual testing of the OpenCart Demo E-commerce Web Application.
It includes test planning, test scenario design, test case writting, defect reporting, and RTM.

## Modules Covered
- User Registration
- Login/Logout
- Product Search
- Product Details Page
- Add to Cart
- Checkout
- Order Confirmation

 ## Testing Types
 - Functional Testing
 - Smoke Testing
 - Regression Testing
 - UI Testing

 ## Tools Used
 - Manual Testing Techniques
 - MS Excel / Google Sheets
