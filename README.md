# Manual Testing Project - OpenCart Demo E-Commerce Website

## Application Under Test
OpenCart Demo E-Commerce Website  
URL: https://demo.opencart.com/

## Project Overview
This project demonstrates end-to-end manual testing of the OpenCart Demo E-Commerce web application.

The testing process was performed using an Agile-based workflow, starting from Epics and User Stories to test scenario creation, test case design, and defect reporting.

Test management and documentation were maintained using Jira and Excel/Google Sheets.

The project includes:
- Test Planning
- User Story Analysis
- Test Scenario Design
- Test Case Writing
- Defect Reporting
- Requirement Traceability Matrix (RTM)

## Modules Covered
Testing was organized module-wise using Epics and User Stories for better traceability and test management.

The following modules were covered:

- Homepage (UI & Navigation Validation)
- Header & Navigation Components
- Category & Product Listing Page (PLP)
- Product Details Page (PDP)
- Product Compare
- Shopping Cart (Pre-Checkout Flow)
- User Authentication (Registration, Login & Logout)

## Testing Types Covered
- Functional Testing
  - Positive Testing
  - Negative Testing
  - Edge Case Testing

## Tools & Technologies Used
- Jira (Bug Tracking & Test Management)
- MS Excel / Google Sheets (Test Documentation)
- GitHub (Project Repository)
- Manual Testing Techniques
